# Dossier Public

Placez ici les ressources statiques de l'application.

## Logo AEM

Pour que la carte fonctionne correctement, vous devez ajouter le logo de l'Association des Étudiants Mauritaniens au Maroc :

1. Renommez votre logo en `logo-aem.png`
2. Placez-le dans ce dossier (`public/logo-aem.png`)
3. Le logo sera automatiquement chargé par l'application

**Spécifications recommandées :**
- Format : PNG (avec transparence de préférence)
- Dimensions : 100x100 pixels minimum
- Aspect ratio : Carré (1:1)
- Taille : < 500KB
